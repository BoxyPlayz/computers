Debian

Base Setup

  • 1. Setup Debian as normal until Software Selection
  • 2. Disable all on Software Select and continue
  • 3. After reboot, you can install any of the following desktop environments (it isn't required):

GNOME Desktop

  • 1. Enter root user by running su
  • 2. Update Packages with apt update
  • 3. Install the desktop environment with
    apt install gnome-shell gnome-control-center gnome-terminal

Software Store

If you desire, you can install a graphical software manager with

apt install gnome-software
